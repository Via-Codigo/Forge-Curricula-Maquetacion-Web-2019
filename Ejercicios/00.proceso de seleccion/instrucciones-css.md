## Como se escribe una regla de CSS

```
<selector> {
    <propiedad>: <valor>;
    <propiedad2>: <valor2>;
}
```

## Propiedades y valores: 

```css
nombre_del_elemento {
    background-color: red; /* El valor es el nombre del color */
    color: red;
    margin:0px; 
    padding:0px;
    border:1px solid red;
    text-align:center;
    font-size:16px;
    text-decoration:none;
    font-style: italic;

    line-height: 1;

    width: 10px;

}
```
